grid

Grid system

Getting started

Installing

npm install @thoughtsunificator/grid

Usage

Static

You can build the grid all at once:

import { Grid } from "@thoughtsunificator/grid"

const grid = new Grid()
grid.build(4)

Dynamic

You could also build the grid cell by cell:

import { Grid } from "@thoughtsunificator/grid"

const grid = new Grid()
grid.addCell(0, 0)
grid.addCell(0, 1)
